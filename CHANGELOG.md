# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-29

### Added
- Initial release of SPY Volatility Analysis
- Realized volatility calculation (close-to-close)
- Parkinson volatility estimator (high-low based)
- Garman-Klass volatility estimator (OHLC based)
- Average True Range (ATR) calculation
- Year-by-year volatility comparison visualization
- Rolling 30-day volatility chart
- Risk-return scatter plot
- Daily return distribution analysis
- Comprehensive summary statistics output
- CSV export of annual metrics
- High-resolution PNG chart output (300 DPI)

### Features
- Analyzes 10 years of SPY data (2015-2025)
- Multiple volatility estimators for robust analysis
- Professional multi-panel visualization dashboard
- Automated calculation of key statistics
- Clean, documented, and modular code structure

### Documentation
- Comprehensive README with usage instructions
- CONTRIBUTING guide for community contributions
- MIT License for open source use
- GitHub Actions workflow for CI/CD
- Setup script for easy environment configuration
